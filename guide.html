Commandes du Bot Discord

!help

Description: Affiche la liste de toutes les commandes disponibles.

Usage: !help

!ban [@utilisateur]

Description: Bannit un utilisateur du serveur.

Usage: !ban @utilisateur

!kick [@utilisateur]

Description: Expulse un utilisateur du serveur.

Usage: !kick @utilisateur

!mute [@utilisateur]

Description: Rend un utilisateur muet dans tous les salons vocaux.

Usage: !mute @utilisateur

!unmute [@utilisateur]

Description: Réactive le micro d'un utilisateur.

Usage: !unmute @utilisateur

!clear [nombre]

Description: Supprime un nombre spécifié de messages dans un salon texte.

Usage: !clear 10

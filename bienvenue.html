Commandes du Bot Discord

/setwelcome

Description: Configurer le canal et le message de bienvenue.

Usage: >/setwelcome

/setleave

Description: Configurer le canal et le message d'au revoir.

Usage: /setleave

/setjoin

Description: Configurer le canal et le message de rejoindre.

Usage: /setjoin

/disablewelcome

Description: Désactiver le message de bienvenue.

Usage: /disablewelcome

/disableleave

Description: Désactiver le message d'au revoir.

Usage: /disableleave

/welcomevars

Description: Afficher les variables disponibles pour les messages.

Usage: /welcomevars

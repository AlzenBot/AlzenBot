Commandes du Bot Discord

a!setwelcome

Description: Configurer le canal et le message de bienvenue.

Usage: >a!setwelcome

a!setleave

Description: Configurer le canal et le message d'au revoir.

Usage: a!setleave

a!disablewelcome

Description: Désactiver le message de bienvenue.

Usage: a!disablewelcome

a!disableleave

Description: Désactiver le message d'au revoir.

Usage: a!disableleave

!unmute [@utilisateur]

Description: Réactive le micro d'un utilisateur.

Usage: !unmute @utilisateur

!clear [nombre]

Description: Supprime un nombre spécifié de messages dans un salon texte.

Usage: !clear 10

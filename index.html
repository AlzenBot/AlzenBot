Guide d'utilisation

Instructions pour inviter et configurer le bot sur votre serveur Discord.

Étape 1 : Invitation du Bot

Pour utiliser NomDuBot sur votre serveur Discord, suivez ces étapes :

  1. Ouvrez Discord et connectez-vous à votre serveur.
  2. Accédez à la page d'invitation du bot.
  3. Cliquez sur le bouton d'invitation et suivez les instructions.
  4. Autorisez le bot à accéder à votre serveur.
Inviter le bot maintenant

Étape 2 : Configuration

Une fois que le bot est invité sur votre serveur, configurez-le comme suit :

  1. Ouvrez Discord et allez dans les paramètres de votre serveur.
  2. Sélectionnez la section "Bot" dans le menu.
  3. Localisez NomDuBot dans la liste des bots autorisés et configurez les permissions selon vos besoins.
  4. Sauvegardez les modifications.

Étape 3 : Utilisation

Maintenant que NomDuBot est invité et configuré, commencez à l'utiliser :

  • Tapez !commande pour utiliser les fonctionnalités du bot.
  • Explorez les différentes commandes disponibles.
  • Consultez la documentation pour des informations détaillées sur chaque commande.
